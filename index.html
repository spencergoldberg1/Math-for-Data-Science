Project 1

Exploratory data analysis and visualization

In this project, I used Python to perform exploratory data analysis and visualization on a dataset of student grades. The goal was to gain insights into factors that affect student performance and identify patterns in the data. I used matplotlib and seaborn libraries to create various visualizations, including scatterplots, boxplots, and histograms.

Skills demonstrated: Python programming, data manipulation and visualization, exploratory data analysis

Project 2

Linear regression and hypothesis testing

In this project, I used Python to perform linear regression and hypothesis testing on a dataset of customer spending at a grocery store. The goal was to identify factors that have a significant impact on customer spending and develop a regression model to predict customer spending. I used scikit-learn library to perform linear regression and hypothesis testing, and evaluated the model using various metrics such as R-squared and mean squared error.

Skills demonstrated: Python programming, data analysis, linear regression, hypothesis testing

Project 3

Machine learning and classification

In this project, I used Python to perform machine learning and classification on a dataset of credit card transactions. The goal was to develop a model to predict fraudulent transactions and evaluate the model's performance using various metrics such as precision, recall, and F1 score. I used scikit-learn library to perform machine learning and classification, and implemented various algorithms such as logistic regression, decision trees, and random forests.
